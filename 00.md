## git

```bash
$ git checkout master

# 先在gitee和github上新建空项目 vue_laravel_tutorial
$ git remote add gitee git@gitee.com:advance/vue_laravel_tutorial.git
$ git push -u gitee master

$ git remote add github git@github.com:youshengyouse/vue_laravel_tutorial.git
$ git push -u github master
```
